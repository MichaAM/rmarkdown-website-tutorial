school-learning environment

Introduction

In this analysis there are 3 models for each school-learning environment analysis: 1. Model 1: General DiD with clustered SE at school level 2. Model 2: DiD with school and year fixed effects 3. Model 3: DiD with school and year fixed effects and additional variables

1. Using computers in language classes

Q: “Do you use a computer during language lessons?”

DiD regression with clustered SE - Jewish sector

didreg.lang.jew <- lm_robust(Lang ~ treat + after + did,
